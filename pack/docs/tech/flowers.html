前端实现 - 木棉花落

触虐の羊明2023-09-16
1413 字 预计阅读 4.7 分钟

这是已经在首页的白天模式实装的特效，这里大概介绍一下它的实现代码。

# Flowers.vue 绘制花瓣飞舞

采用canvas画布实现。

花瓣分为前、后两个图层，z-index分别为10和-10，各用一个画布。

<template>
  <div id="flower-canvas">
    <canvas ref="frontCanvas" :width="w" :height="h" class="flower-canvas front">
    </canvas>
    <canvas ref="backCanvas" :width="w" :height="h" class="flower-canvas back">
      您的浏览器不支持canvas请升级！
    </canvas>
  </div>
</template>

css样式如下：

<style scoped>
.flower-canvas {
  position: fixed;
  top: 0;
  left: 0;
  pointer-events: none;
}

.front {
  z-index: 10;
  filter: blur(1px);
}

.back {
  z-index: -10;
  filter: blur(.25px);
}
</style>

# 脚本实现

组件定义了两个父级传入属性：当前的风力(windStrength)和风向(windAngle)，会影响花瓣的速度。

ts脚本部分概括如下：

<script setup lang="ts">
import { ref, onMounted } from 'vue';
import { isMobile, inRange, randElem, randFloat, type Range } from '../utils.ts';

const frontCanvas = ref<HTMLCanvasElement>();
const backCanvas = ref<HTMLCanvasElement>();

const props = defineProps({
  windStrength: {
    type: Number,
    default: 0
  },
  windAngle: {
    type: Number,
    default: 0
  }
});

// canvas画布的宽高，受resize事件响应式变动
const w = ref(1920);
const h = ref(280);

// 花瓣类，定义每一片花瓣的共同属性
class Flower {
    width: number; 
    height: number;
    x: number; 
    y: number; 
    rotate: number;
    speedX: number; 
    speedY: number; 
    speedR: number;
    scale: number;
    img: HTMLImageElement | null;
    constructor(
        public scaleRange: number[] | Range,
        public speedXRange: Range,
        public speedYRange: Range
    ); //构造函数初始化

    // 初始化加载图片
    show(ctx: CanvasRenderingContext2D);

    // 重置花瓣位置
    regen(dx: number, dy: number, rand: boolean = false);

    // 每帧花瓣绘制逻辑
    draw(ctx: CanvasRenderingContext2D, rotate: number = 0);

    // 每帧花瓣移动逻辑，会受风的影响
    move(ctx: CanvasRenderingContext2D);
}

onMounted(() => { ... })
</script>

# Flower类实现

图片加载方法。

Flower.show

show(ctx: CanvasRenderingContext2D) {
    this.img = new Image();
    // 随机图片
    const src = Math.random() > 0.5 ? "/images/flower1.svg" : "/images/flower2.svg";
    this.img.src = src;

    this.regen(this.speedX, this.speedY, true);

    // 画初始位置的花瓣
    this.img.onload = () => {
        const ratio = this.img!.width / this.img!.height; // 宽高比，用于缩放
        [this.width, this.height] = [ratio * this.scale, this.scale];
        ctx.drawImage(this.img!, this.x, this.y, this.width, this.height);
    }
}

花瓣位置的初始化方法，当花瓣加载和飞出屏幕时会调用。

Flower.regen

regen(dx: number, dy: number, rand: boolean = false) {
    // 随机大小
    this.scale = Array.isArray(this.scaleRange)
        ? randElem(this.scaleRange)
        : randFloat(this.scaleRange);
    [this.speedX, this.speedY] = [randFloat(this.speedXRange), randFloat(this.speedYRange)];
    // 随机初位置
    if(rand) {
        [this.x, this.y] = [randFloat(0, w.value), randFloat(0, h.value)]
    } else {
        // 根据宽高比、当前速度决定其随机分别在水平边界、垂直边界上生成的概率
        const ratio = (w.value + dy) / (w.value + h.value + dx + dy);
        const [initX, initY] =  Math.random() < ratio
        // 根据当前速度正交分量的正负，决定其在垂直边界的左/右、水平边界的上/下生成
            ? [randFloat(0, w.value), dy >= 0 ? 0 : h.value]
            : [dx >= 0 ? 0 : w.value, randFloat(0, h.value)];
        [this.x, this.y] = [initX, initY];
    }
}

花瓣的每帧绘制方法，包含旋转。

Flower.draw

draw(ctx: CanvasRenderingContext2D, rotate: number = 0) {
    if(!this.rotate) {
        return ctx.drawImage(this.img!, this.x, this.y, this.width, this.height);
    }
    ctx.save();
    ctx.translate(this.x, this.y);
    ctx.rotate(rotate * Math.PI / 180);
    ctx.drawImage(this.img!, 0, 0, this.width, this.height);
    ctx.restore();
}

花瓣的每帧移动逻辑。

Flower.move

move(ctx: CanvasRenderingContext2D) {
    // 获取当前风力和风向，与花瓣固有速度求和，计算dx和dy
    const wStr = props.windStrength, wAgl = props.windAngle * Math.PI / 180;
    const dx = this.speedX + (wStr ? wStr * Math.cos(wAgl) : 0);
    const dy = this.speedY + (wStr ? wStr * Math.sin(wAgl) : 0);
    const { width: cw, height: ch } = this;
    // 当花瓣飞出屏幕时，重置其位置，否则改变位置
    if (!inRange(this.x + dx, -cw, w.value + cw) || !inRange(this.y + dy, -ch, h.value + ch)) {
        this.regen(dx, dy);
    } else [this.x, this.y] = [this.x + dx, this.y + dy];
    this.rotate = (this.rotate + this.speedR) % 360;
    // 绘制
    this.draw(ctx, this.rotate);
}

# 挂载初始化

组件挂载时，对每个图层初始化花瓣飞舞canvas画布。

onMounted(() => {
  const scales = [[10, 12.5, 15], [17.5, 20, 22.5]];
  const speedXRange = { min: 0, max: 2 };
  const speedYRange = { min: 1, max: 1.5 };
  const FlowerCounts = isMobile(window) === false ? [20, 5] : [10, 3];
  w.value = window.innerWidth;
  h.value = window.innerHeight;

  for (const [i, canvas] of [backCanvas.value, frontCanvas.value].entries()) {
    if (!canvas) return;
    let ctx = canvas.getContext('2d');
    if (!ctx) return;

    const flowers: Flower[] = [];
    for (let t = 0; t < FlowerCounts[i]; t++) {
      const fr = new Flower(scales[i], speedXRange, speedYRange);
      flowers.push(fr);
      fr.show(ctx);
    }

    // 20ms一帧，控制画布内所有花瓣的移动
    function movee() {
      if (!ctx) return;
      ctx.clearRect(0, 0, w.value, h.value);
      for (const flower of flowers) {
        flower.move(ctx);
      }
    }
    setInterval(movee, 20);
    movee();

    // 监听窗口大小改变事件
    window.addEventListener("resize", () => {
      w.value = window.innerWidth;
      h.value = window.innerHeight;
    });
  }
})

# WindBase.vue 随机风力产生

Flowers组件被装载在一个名为WindBase的提供随机风力的组件中。该组件会为插槽(slot)中的组件提供统一的风力管理。

<template>
  <div class="wind-base">
    <slot :strength="windStrength" :direction="windAngle" />
  </div>
</template>

通过插槽(slot)机制，可直接将WindBase所生成的当前风力(windStrength)和风向(windAngle)参数传入Flowers组件中。

目前只有Flowers会受风力调制，以后可能还会TODO其它的特效，比如下雨。

在父组件中：

<template>
  ...
  <WindBase>
    <template #default="{ strength, direction }">
      <Flowers v-if="!isDarkMode" :wind-strength="strength" :wind-angle="direction" />
    </template>
  </WindBase>
</template>

至于风力(windStrength)和风向(windAngle)参数如何生成，这是由WindBase组件的脚本所实现。

组件会在一段时间内概率（此处为每5秒有20%的概率）触发一次风力的改变。

<script setup lang="ts">
import { ref, watch, onMounted } from 'vue';
import { isMobile, randFloat, randInt, passTo } from '../utils.ts';
import Flowers from './Flowers.vue';

// 风力风向
const windStrength = ref(0);
const windAngle = ref(180);
// 平滑过渡
const windStrengthTo = ref(0);
const windAngleTo = ref(180);
const windStep = ref(1);

let itv: NodeJS.Timeout | null = null;

// 监测风力风向的变化，提供风力改变的平滑过渡
watch([windStrengthTo, windAngleTo], (nvals) => {
    itv = setInterval(() => {
        windStrength.value = passTo(windStrength.value, nvals[0], windStep.value);
        windAngle.value = passTo(windAngle.value, nvals[1], windStep.value);
        if(windStrength.value === nvals[0] && windAngle.value === nvals[1]) {
            clearInterval(itv!);
        }
    }, 20);
})

onMounted(() => {
    // 随机改变风力风向（仅在电脑端启用）
    if(isMobile(window) === false) { 
        setInterval(() => {
            if(Math.random() < 0.2) {
                itv && clearInterval(itv);
                windStrengthTo.value = randFloat(0, 4);
                windAngleTo.value = Math.random() < 0.5 ? randInt(-10, 30) : randInt(150, 190);
                windStep.value = randFloat(0.1, 0.5); // 风向改变的加速度
            }
        }, 5000);
    }
})
</script>

其中passTo函数很容易实现，就是将参数val向目标参数dest的方向移动step值。

当windStrengthTo和windAngleTo改变时，现有的windStrength、windAngle的值将会自动向它们移动。

export function passTo (val: number, dest: number, step: number = 1): number {
    if(dest < val) {
        val = Math.max(val - step, dest);
    } else if(dest > val) {
        val = Math.min(val + step, dest);
    }
    return val;
}

页面规划

触虐の羊明2023-10-25
782 字 预计阅读 2.6 分钟

以下是本网站目前的一些页面功能需求。

# 画册（开发中）

路由: /albums

对应QQ空间的相册，这里是放我平日里的摸鱼、世界观的原画图设，以及画画参考的地方。 当然，实现的底层机制和文章类的页面一样，都是有markdown正文的页面。

# 文章

路由: /docs

这里是放正式文章的，与QQ空间日志的不同在于，这里的文章必须有一定的实际内容。 我会用来放世界观的文字设定、剧情故事（如果有的话）、文化研究及科普、一些技术分享等等。

# 随想

路由: /posts

对应QQ空间的日志功能。 放的是我平日里的想法、随笔、未经整理的脑洞等，和我在QQ空间置顶日志里放的内容差不多（甚至同步更新）。 内容比较随意。当然，也难免会有一些负能宣泄。

# 说说（开发中）

路由: /board

对应QQ空间的说说。 这里与前面最大的不同在于，说说不只是我可以发，注册用户都可以发，相互交流。相当于一个留言板，底层是加强版的评论区。 计划实现配图、配视频，以及点赞、转发功能，至于分享……我再看看怎么实现吧。

# 世界观（规划中）

路由: /world

等做完了相册和说说功能以后，就得开始做OC的世界观介绍相关的页面了，起码最早也要包括人设介绍和地方介绍，到时候就不只是代码需求了，美术需求也很重。

啊……哪有人既干美工的活又打代码的……甚至连……文案和编剧都得靠自己。脑子和时间都不够用，恨不得分身，我还有画画班要上啊……

甲方(我自己)的需求是：

世界观页面，至少包括角色和地方介绍，参考二游的官方网站。

要有与它们“属于同一类”的既视感。核心目的是“接轨”和“引导”——在第一印象上，先给用户一股二次元和异世界相关亚文化的联想，页面内容与表现做出ACG的浓度至关重要。这是让内容文化题材，在纳入思考之前尝试洗脱那些在官方宣传里所遗留的“土味”这种固有debuff印象的关键一步。“有没有人看，破不破得了圈”全看的是这一步。
